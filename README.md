# DIAZ — AI Chatbot

A premium, Apple-inspired AI chatbot built with **pure HTML, CSS, and JavaScript**.
Created as part of the **CodeAlpha Internship** program.

---

## ✨ Features

- 🎨 Premium Apple-inspired UI (dark black theme with cyan accents)
- 📱 Fully responsive — looks great on mobile, tablet, and desktop
- 💬 30+ predefined intelligent responses (greetings, jokes, quotes, tech, time, date, and more)
- ⌨️ **Enter key** to send and a working **Send** button
- 🕘 Persistent **chat history** via `localStorage`
- 🔽 Smooth **auto-scroll** to the latest message
- ⌛ Animated typing indicator
- 🧹 One-click **Clear chat** option
- 🪶 Zero dependencies — vanilla JavaScript only

## 🚀 Getting Started

1. Download or clone this folder.
2. Open `index.html` in any modern browser (Chrome, Safari, Edge, Firefox).
3. Start chatting with **DIAZ**!

No build tools, no installations — just open and run.

---

## 🧠 How It Works

- User input is matched against a list of **keyword rules** in `script.js`.
- Each rule returns one of several possible responses (randomized for variety).
- If no rule matches, DIAZ falls back to a friendly default reply.
- All messages are saved to `localStorage` so your conversation persists across reloads.
