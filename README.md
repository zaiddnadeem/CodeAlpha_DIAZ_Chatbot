DIAZ AI Chatbot

DIAZ is a responsive AI chatbot built using HTML, CSS, and JavaScript.

Features

- Responsive mobile-friendly design
- Modern dark UI
- Chat history support
- Predefined intelligent responses
- Enter key support
- Send button support
- Typing animation
- Clear chat option

Technologies Used

- HTML5
- CSS3
- JavaScript

How to Run

1. Download the project files.
2. Open index.html in a web browser.
3. Start chatting with DIAZ.
